# Code of Conduct

We are committed to providing a welcoming and respectful community. All contributors and participants are expected to follow the guidelines below.

Expected behavior

- Treat everyone with respect and courtesy.
- Accept feedback and technical critique constructively.
- Use inclusive language.

Unacceptable behavior

- Insults, harassment, discriminatory or intimidating language.
- Threats, stalking, or publishing private information.
- Sexist, racist, or otherwise abusive personal attacks.

Reporting an incident

If you are the victim or witness of unacceptable behavior, please contact the project maintainers by opening an issue titled "Code of Conduct Incident" or contact a maintainer directly via GitHub.

References

- This project follows common open source practices. For a full and official text, consider the Contributor Covenant: https://www.contributor-covenant.org/ (version 2.1).
